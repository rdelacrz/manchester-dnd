# SRD 5.1 content boundary

This directory is reserved for normalized records sourced from the Creative Commons release of SRD 5.1. No rules prose has been imported yet.

Every future record must include:

- `ruleset_id: srd-5.1-cc`;
- source document version and section/page provenance;
- whether wording is verbatim or adapted;
- a stable local identifier that does not depend on display text.

Do not populate this directory from the 2018 Basic Rules PDF or from commercial rulebooks. Keep original setting, themes, and event prompts outside this directory so provenance remains auditable.
